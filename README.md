# Binify

Binify is a school team project focused on **IoT (Internet of Things)**.  
The goal was to design and implement a useful device that collects real-world data and processes it in a distributed system.

## Project goal

The team decided to create a smart device based on **Raspberry Pi / Raspberry Pi Pico**, designed to measure:
- the **fullness** of waste containers
- the level of **odor intensity** (“smelliness”)

## Project structure

The project consists of multiple parts:
- **Firmware** running on the embedded device (Raspberry Pi Pico, Zig)
- **Backend services** for data collection, processing, and synchronization
- **Frontend application** for data visualization
- **Cloud components** for deployment and integration
